Computer Network Practicals
This repository contains all the experiments performed in the **Computer Networks Laboratory** course.
Each practical focuses on understanding and implementing key networking concepts, from basic elements to advanced routing algorithms.


 List of Practicals

|   Practical No.   |   Title of Practical                                                                                 |
| :---------------: | ---------------------------------------------------------------------------------------------------- |
|         1         | Study of basic elements of computer networking with details of networking devices.                   |
|         2         | Packet analysis using a packet tracer tool with detailed analysis of Application Layer packets.      |
|         3         | Socket Programming for Transport Layer packets (TCP and UDP).                                        |
|         4         | Analysis of Application Layer packet (DNS) using DNS commands & analysis using a packet tracer tool. |
|         5         | Implement Sliding Window Protocols.                                                                  |
|         6         | Implement Traffic Shaping Algorithms (Leaky Bucket, Token Bucket).                                   |
|         7         | Implement Distance Vector Routing Algorithm.                                                         |
|         8         | Implement Link State Routing Algorithm.                                                              |
|         9         | Simulation of Congestion Control Algorithms.                                                         |
|         10        | Study of Network Simulation using NS2 / Wireshark.                                                   |

---

Tools & Technologies Used
* **Cisco Packet Tracer** – for packet-level simulation
* **Wireshark** – for live packet capture and protocol analysis
* **Python / C** – for socket programming and protocol implementation
* **Command Line Tools** – for DNS and networking utilities


Learning Outcomes
* Understand OSI and TCP/IP layer interactions
* Perform protocol-level packet analysis
* Implement transport and network layer algorithms
* Gain hands-on experience with traffic shaping and routing
